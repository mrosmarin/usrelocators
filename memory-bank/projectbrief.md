# Project Brief

## Project
Nobody Move CRM platform (`quiet-carry-core`), a Vite + React + TypeScript application with consumer, professional CRM, and admin-oriented workflows.

## Primary Goal
Deliver an execution-ready MVP aligned to the business specification and internal implementation docs, with clear backlog structure and phased rollout.

## Current Confirmed Scope Focus
- 12-stage journey migration across consumer + CRM surfaces.
- Activation Card and Branded Sheets operational completion/hardening.
- Linear backlog execution via team `Nobodymove` with story points, `P0/P1/P2/P3` priority model, unassigned by default.

## MVP Delivery Streams
1. Foundations (auth, RBAC, auditability)
2. Consumer onboarding + journey
3. Professional CRM workflows
4. Docs/content + decision support tools
5. Stress alerts + intelligence behaviors
6. Vendor/care operations
7. Integrations (Stripe, OAuth, email, Calendly, Claude)
8. QA/pilot readiness

## Non-Goals (Current)
- Treat generic Lovable README as source-of-truth for architecture.
- Broad V1.5/V2.0 expansion before MVP critical path and pilot readiness are complete.
